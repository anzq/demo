# demo
使用Vue全家桶高仿饿了么的一个demo,使用node做中间件拉取饿了么数据包,使用mongoDB做的登陆注册

### 技能栈
##### Vue
##### Vuex
##### vue-router
##### axios
##### node
##### mongoDB


#安装步骤
```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
####温馨提示node也要启动服务哦！
```
# install dependencies
npm install

# serve with hot reload
npm start

```
